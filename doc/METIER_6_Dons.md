# Définition d'un don

# Données communes à tous les dons
* un don est un capacité spéciale qui vient en complément des aptitudes raciales et des aptitudes de classe.
* chaque sort appartient à une catégorie. Les catégories de dons sont spécifiques à la ruleset
* un don est décrit par :
    - un nom
    - une description 
    - une catégorie

# Ruleset DD3.5
* un personnage gagne un don soit à un niveau indiqué par la table de progression de la classe dans laquelle il vienet de gagner un niveau, soit selon la table de progression générale (en remplacement de l'augmentation de caractéristiques)

# Ruleset DD204
* un personnage gagne un don sgrace à son historique soit selon la table de progression générale (en remplacement de l'augmentation de caractéristiques)
