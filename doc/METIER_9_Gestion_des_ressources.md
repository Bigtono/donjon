## Note importante : filtrage des données selon la ressource
 1. Les règles de jeu sont issues de différents livres et supplément de règles regorupés sous le terme ressource.
 2. Quelque soit la situation, les classes, races, sorts, dons et objets magiques sont toujours filtrés selon leur ressource via le champ [prefixe de table]_res_id présent dans chacune de ces tables
 3. La liste des id des ressources sélectionnées est gérée par la page affichageSelectionSources.php. Cette page doit être incluse dans le code à chaque que le filtrage par ressource est nécessaire
