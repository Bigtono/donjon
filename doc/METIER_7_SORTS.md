# Définition d'un sort

# Données communes à tous les sorts
* un sort est rattaché à un niveau de sort allant de 0 à 9
* chaque sort appartient à un collège de magie, aussi appelé école
* un sort est inscrit dans la liste de sorts de une ou plusieurs classes de personnage.

# Ruleset DD3.5
* un sort peut posséder une branche et un registre. Le registre, quand il existe, est présenté entre crochets
