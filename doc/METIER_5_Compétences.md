# Définition d'une compétences
Les compétences représentent toute une palette d'aptitudes différentes. Un personnage maîtrise ses compétences de mieux en mieux à mesure qu'il monte de niveau. 

# Données communes à tous les compétences
* Une compétence est décrite par :
    - un nom
    - une caractéristique associée
    - une description 
    - une ressource
    - un ruleset 
Le bonus de la caractéristique associée est ajoutée a ujet de dé lors d'un test de compétence

# Ruleset DD3.5
Un personnage gagne des points de compétences à chaque fpois qu'il monte de niveau. Le nombre de points gagnés est fonction de la classe dans lequel le personnage décide de monter de niveau. Il peut affecter ces points à des compétences pour gagner des rangs de maîtrise.

Si une compétence figure dans la liste des "compétences de classe" de la classe dans lequel le personnage vient de gagner un niveau, le personnage gagne 1 rang de maîtrise dans la compétence pour chaque point dépensé. Si la compétence ne figure pas dans cette liste, le personnage doit dépenser deux points pour gagner un rang dnas la compétence.

Ce ruleset introduit plusieurs nouvelles notions pour les compétences
* Malus d'armure : le fait de porter une armure autre qu'une armure de cuir implique un malus au jet pour cette compétence. La valeur du malus est précisée dans le descriptif de l'armure en question
* Formation nécessaire : si la formation est nécessaire, un personnage ne peut pas utiliser la compétence sans avoir été formé préalablement ()
